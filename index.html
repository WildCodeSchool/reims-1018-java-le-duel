Voici notre blog afin de vous faire suivre l'actualité de notre projet, disponible sur GitHub : reims-1018-java-le-duel.

Bienvenue sur le blog representant le projet "Duel de sorciers" portant le nom de "Tavern of the dashing Goblin". Ce projet a été réalisé dans le cadre de la formation "Devellopeur Web" au sein de la WILDCODE School de Reims. Notre groupe s'est nommé "House Dashing Goblins" (Maison des Gobelins Fringuants), et qui est composé de : Erwan GENEVE, Tantely RANDRIA, Marthe CHARPY et Mohamed-Kamel LAHRI. Ce projet qui aura duré 6 semaines, aura pour but dee vous presenter, un jeu sur le thème d'HARRY POTTER, qui mettra en avant un systeme du duel individuel contre l'IA, ou un systeme de duel contre un autre joueur. Un projet que nous n'avons pas travaillés a temps plein dû a l'apprentissage du language de programmation JAVA / JEE, et des autres outils de programmation informatique. Le projet a été modelé sur 3 semaines de programmation JAVA sur le terminal, puis 3 semaines de programmation sur une interface graphique, afin de pouvoir offrir un accès simplifié à n'importe quel utilisateur. Hormis les logiciels specifiques au traitement de texte pour le code, nous avons utilisés Github pour l'echange de fichiers simplifiés et l'hebergement de ce blog, La Methode SCRUM avec Trello qui utilise un tableau sous forme de sprint nous permettant d'identifier les priorités hebdomadaires dit de "Sprint" et donc de faire un point en fin de semaine sur ce qui a été effectué et ce qu'il reste a faire. En naviguant sur les onglets "Sprint", vous decouvrirez le parcours que nous avons effectués afin d'arriver au resultat qu'est ce jeu aujourd'hui.

Product by: Marthe Charpy - Erwan GENEVE - Mohamed-Kamel LAHRI - Tantely RANDRIA.
